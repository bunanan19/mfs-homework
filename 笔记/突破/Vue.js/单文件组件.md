```shell
vue init webpack vue-start
```

2. [[15\]3Sum](https://leetcode.com/problems/3sum)

   **三数之和**

   给你一个 n 个整数的数组S，能找到 a,b,c 属于 S，使得 `a + b + c = 0`吗？请找出所有的唯一的三元组，使得他们的和为0

   **注意：**你的解不能包含重复的三元组

   ```
   如给出 S = [-1, 0, 1, 2, -1, -4]
   
   则解是
   [
      [-1, 0, 1],
      [-1, -1, 2]
   ]
   ```
